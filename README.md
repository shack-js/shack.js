# shack.js

with shack.js you can just forget `fetch`, `axios` and things alike, you call backend functions directly, and of course with access control


## quick usage

```
npx @shack-js/cli init my-app 
cd my-app
npm i
npm run dev
# npm run build
# npm run start
```
